# [백준 - 실버 2 ] 도영이가 만든 맛있는 음식 (2961 번)

## ⏰  **time**

10분

## :pushpin: **Algorithm**

- 브루트포스 알고리즘 
- 비트마스킹 
- 백트래킹

## ⏲️**Time Complexity**

$O()$

## :round_pushpin: **Logic**

- 신맛(Sour)은 곱, 쓴맛(Bitter)은 합, 최소화할 값은 abs(신맛 - 쓴맛)
- 공집합(아무 재료도 안 쓰는 경우)은 금지 → 최소 1개 이상 선택 
- 백트래킹으로 현재까지의 (곱, 합)을 들고 가면서 해당 재료를 선택하는 경우는 `s *= S[i], b += B[i]`, 미선택하는 경우는 그대로 진행 한다. 
- 그리고 한 개 이상 선택된 상태에서만 답 후보 갱신한다.

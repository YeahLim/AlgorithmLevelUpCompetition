# [백준 - 골드 4] 회사 문화 1 (14267)
 
## ⏰  **time**

20분

## :pushpin: **Algorithm**

DP, BFS

## ⏲️**Time Complexity**

$O(N)$

## :round_pushpin: **Logic**
1. 문제풀이
   
   현재 사원이 칭찬을 받으면 하위 사람은 모두 현재 사원이 칭찬을 받은 것을 모두 추가해야한다.

   그러므로 점화식은 다음과 같다.

```
   칭찬을 받은 횟수[다음] = 칭찬을 받은 횟수[현재] + 칭찬을 받은 횟수[다음]
```

## :black_nib: **Review**
- 누적합의 규칙을 트리에 적용하면 DP로 풀면 된다.

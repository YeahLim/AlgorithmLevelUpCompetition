## :round_pushpin: **Logic**
- 주문 순서대로 넣어야 하는데 보조 컨테이너는 가장 최근에 넣은것만 뺄 수 있으므로 스택으로 구현한다
- 일단 보조컨테이너에 넣고 보조컨테이너에서 최근에 넣은 것 부터 한개씩 꺼내면서 주문순서에 맞는것만 넣으면 코드가 간결해진다

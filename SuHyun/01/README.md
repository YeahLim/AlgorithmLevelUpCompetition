# [백준 - 골드 4] 직사각형으로 나누기 (1451)
 
## ⏰  **time**

시간 초과

## :pushpin: **Algorithm**

브루트포스, 누적합

## ⏲️**Time Complexity**

$O(N^2)$

## :round_pushpin: **Logic**
1. 문제풀이

주어진 크기에서 3개의 직사각형을 만들수있는 경우의 수는 3가지다.

1. 가로 3개
2. 세로 3개
3. 왼쪽 1개 오른쪽 2개
4. 왼쪽 2개 오른쪽 1개
5. 위 1개 아래 2개
6. 위 2개 아래 1개

각 경우의 수를 모두 구해 2차원 누적합을 사용하여 영역의 수의 합을 구한 뒤 모두 곱한 값의 최대값을 구하면 된다.

## :black_nib: **Review**
- 일단 어떤 모양이 나오는지 모든 경우의 수를 확인해보자

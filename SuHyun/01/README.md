# [백준 - 골드 4] 팀 빌딩 (22945)
 
## ⏰  **time**
20분

## :pushpin: **Algorithm**
투포인터

## ⏲️**Time Complexity**
$O(N)$

## :round_pushpin: **Logic**\

양 끝의 위치를 선정하고 왼쪽 포인터를 오른쪽으로 움직일지 오른쪽 포인터를 왼쪽으로 움직일지 기준을 정하면 된다.

이 때 둘 중 최대값을 옮겨도 둘의 최솟값의 최대치가 바뀌지 않으므로 둘 중 더 작은 값을 움직여 더 크게 만들어야 한다.

# [백준] 구간 합 구하기 4 (11659번)

## ⏰ **time**

10분

## :pushpin: **Algorithm**

구간합 알고리즘

## ⏲️**Time Complexity**

$O(1)$

## :round_pushpin: **Logic**

- 인덱스를 차례대로 모두 더해주면 시간초과가 나기 때문에 리스트를 받고 모든 인덱스의 누적합을 구한 뒤,
  구하는 끝 인덱스 누적합 - 시작 인덱스 직전 인덱스 누적합을 빼주면 된다

## :black_nib: **Review**

- 누적합 연습용으로 오랜만에 다시 풀었다

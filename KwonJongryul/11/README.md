# [백준] 인구 이동 (16234번)

## ⏰ **time**

1시간

## :pushpin: **Algorithm**

- bfs

## :round_pushpin: **Logic**

- 먼저 델타이동 하면서 돌면서 연합들을 체크해 준 후, 숫자를 바꿔준다. 처음엔 연합 이동할때마다 카운터를 셌는데 한번 돌때 전 연합 이동한 숫자를 세야되더라.
  그래서 플래그 세워 준 후, 이번에 한번이라도 이동했으면 처음부터 다시돌면서 검사하고, 아니면 while문 탈출하도록 로직을 짰다

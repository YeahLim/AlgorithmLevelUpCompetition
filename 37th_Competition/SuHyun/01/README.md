# [백준 - 골드 4] 떡 돌리기 (20007)
 
## ⏰  **time**
32분

## :pushpin: **Algorithm**
다익스트라

## ⏲️**Time Complexity**

$O(NlogN)$

## :round_pushpin: **Logic**\

각 집간의 최단거리를 구하면 되므로 다익스트라를 이용하면 된다.

이 때 하루에 X만큼 걸을 수 있고 무조건 왕복으로만 걸어야 하므로 실제 계산거리는 최단거리*2가 된다.

또한 X보다 먼 왕복거리의 집은 도달하지 못하므로 예외를 등록해야한다.

## :black_nib: **Review**
- 노드간의 최단거리 = 다익스트라 알고리즘

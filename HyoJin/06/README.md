## 📍 Logic
두 수를 선택한다 (e : 현재 > s : 기억)

e제곱 - s제곱을 했을 때 G이면 e 추가하고 s+=1 e+=1까지 해줘야함

G보다 작으면 e를 더 큰 수로
                     
G보다 크면 s를 더 큰 수로
                     
연속하는 두 수 제곱의 차 > G이면 종료 (99 100 : 9801 10000)

지난번에는 (현재+기억)(현재-기억) = g 일때 1부터 g까지 돌면서 g의 약수를 target값으로 두었다

# [백준 - 골드 3] 일감호에 다리 놓기

## ⏰  **time**
26분

## :pushpin: **Algorithm**
MST(크루스칼 알고리즘)

## ⏲️**Time Complexity**
$O(N)$

## :round_pushpin: **Logic**
- 모든 인접한 구역은 거리가 0인 도로가 있는 것을 가정한다.
  - 이때 공사중인 곳은 제외한다.
- 공사중인 지역이 1개 이하이면 굳이 가운데 돌을 밟지 않고 모든 곳을 갈 수 있으므로 해당 조건은 계산하지 않는다
- 나머지의 경우는 mst를 사용해 값을 구하고 K를 넘는지 확인한다.

## :black_nib: **Review**
- 엣지 중심은 크루스칼 알고리즘
- 모든 노드를 사용해야 하는지 생각해보자

# [백준 - 골드 3] 학교 탐방하기

## ⏰  **time**
40분

## :pushpin: **Algorithm**
MST(크루스칼 알고리즘)

## ⏲️**Time Complexity**
$O(N^2)$

## :round_pushpin: **Logic**
- 경로 중심으로 최단거리의 합을 구해야 하므로 크루스칼 알고리즘을 사용해야한다.
- 최단거리의 합과 최장거리의 합 2개를 동시에 구해야하므로 우선순위큐도 2개를 만든다.
- 오르막길은 0, 내리막길은 1인걸 유의애햐한다.
- 오르막 길, 내리막 길은 문제의 표현상이고 실제로 방향 그래프가 아닌 무방향 그래프이다. 

## :black_nib: **Review**
- 엣지 중심은 크루스칼 알고리즘

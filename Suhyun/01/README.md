# [백준 - 골드 2] 악덕 영주 혜유

## ⏰  **time**
20분

## :pushpin: **Algorithm**
MST(크루스칼 알고리즘), DFS

## ⏲️**Time Complexity**
$O(N^2)$

## :round_pushpin: **Logic**
- 엣지를 중심으로 최소신장트리를 구해야 하므로 크루스칼 알고리즘을 사용한다.
- N이 최대 1,000이므로 DFS로 모든 노드간 최대 거리를 구해도 (시작 노드 개수) * (다음 노드 개수) = N^2 = 10^6 < 10^8(1억) 이므로 DFS로 노드 간 최대 길이를 구한다

## :black_nib: **Review**
- 엣지 중심은 크루스칼 알고리즘
- 그래프 DFS의 시간 복잡도는 N

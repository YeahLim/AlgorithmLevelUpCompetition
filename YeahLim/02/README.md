# [백준] 치킨 배달 (15686)

## ⏰  **time**

1시간

## :pushpin: **Algorithm**

DFS

## ⏲️**Time Complexity**

$O(2^{N*N})$

## :round_pushpin: **Logic**
- 정석 로직 - 1차원 배열 조합
    `치킨집만 List로 만들어서 순회하며 조합 구하기`
- 내가 쓴 방법 - 2차원 배열 조합
    `2차원 배열인 map을 순회하며, 치킨집일 때만 조합 구하기` 
## :black_nib: **Review**
- 2차원 배열은 DFS로 잘 돌리지 않는데, 간만에 해보았다. 인덱스 실수로 디버깅만 30분 넘게 했다.. 역시 2차원 배열 조합은 피해야한다. 실수가 많이 나고, 코드 보기도 비교적 까다롭다. 왜 생각이 나지 않았을까..!

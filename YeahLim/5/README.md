# [백준] 1, 2, 3 더하기 (9095번)

## ⏰  **time**
15분

## :pushpin: **Algorithm**
1. DFS, 중복순열
2. DP

## ⏲️**Time Complexity**
1. $O(3^N)$
2. $O(N)$

## :round_pushpin: **Logic**
1. DP 초기화
```
dp[1] = 1;
dp[2] = 2;
dp[3] = 4;
```
2. DP 점화식
```
  // i = (1 + i-1) + (2 + i-2) + (3 + i-3)이므로 1,2,3으로 이루어질 수 있는 개수는 아래와 같다
  dp[i] = dp[i-1] + dp[i-2] + dp[i-3];
```

# [백준 - 실버 3 ] 영단어 암기는 괴로워 (20920번)

https://www.acmicpc.net/problem/20920

## ⏰ **time**

20분

## :pushpin: **Algorithm**

자료구조

## ⏲️**Time Complexity**

$O(N)$

## :round_pushpin: **Logic**

1. 먼저 딕셔너리에 단어들이 나온 횟수를 기록해 준다.
2. 그 다음 배열에 딕셔너리들에 키를 넣고 정렬한다.
3. 우선순위가 낮은 순서부터 정렬해 나가면 정답이다.

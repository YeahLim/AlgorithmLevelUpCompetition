# [백준 - 실버 4] 30 (10610번)

## ⏰ **time**

30분

## :pushpin: **Algorithm**

그리디

## ⏲️**Time Complexity**

$O(n)$

## :round_pushpin: **Logic**

1. 내림차순으로 정렬해서 n을 받는다
2. 모든 자릿수의 합이 3의 배수이면 그 수는 3의 배수라고 하니 그에따라 검사해준다.
3. 또한 30의 배수이니 0을 포함하지 않으면 그건 -1을 출력한다.
4. 검사를 모두 통과하면 n, 그렇지 않으면 -1 출력해준다.

## :black_nib: **Review**

- 실버그리디라 얕봤는데 어려웠다. 3의 배수 검사 저건 처음보는 공식이라 더더욱 당황했다...

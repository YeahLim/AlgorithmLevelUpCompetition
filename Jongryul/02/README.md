# [백준 - 실버 5 ] 그룹 단어 체커 (1316번)

https://www.acmicpc.net/problem/1316

## ⏰ **time**

10분

## :pushpin: **Algorithm**

문자열

## ⏲️**Time Complexity**

$O(N^2)$

## :round_pushpin: **Logic**

1. 문자열 하나씩 검사해서 연속된 문자가 아닌데 set안에 있으면 그룹단어가 아니기 때문에, 해당 로직으로 구해준다.

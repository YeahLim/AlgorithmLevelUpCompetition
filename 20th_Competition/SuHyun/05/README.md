# [백준 - 골드 5] 로봇 프로젝트 (3649)
 
## ⏰  **time**

20분

## :pushpin: **Algorithm**

투 포인터

## ⏲️**Time Complexity**

$O(NlogN)$

## :round_pushpin: **Logic**

1. 문제풀이

두 수의 합을 구하는 경우 투포인터를 활용하면 O(2N)의 방법으로 탐색할 수 있다.

이때 테스트케이스의 수가 정해져있기 않기 때문에 입력값이 없을때 까지 입력을 받아야 한다.

## :black_nib: **Review**
- 특정 값의 두 수의 합을 구하는 문제는 투포인터로 해결할 수 있다

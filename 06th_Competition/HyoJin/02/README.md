# [백준] 게임 개발 (1516)
## ⏰ time
오래걸림

## 📌 Algorithm
위상정렬

## 📍 Logic
1. 진입차수가 0인 것들을 큐에 넣는다.
2. 큐에서 한 개를 뽑은 후, (=> 즉 건물을 짓는다. T[i]만큼 시간이 걸린다.)
   1. T[i]와 연결된 다른 건물들 차수를 1씩 줄인다.
   2. result[연결된 다른 건물들] = max(result[현재까지 걸린 시간] + T[i], result[연결된 다른 건물들])
   3. 이때 차수가 0이 되면 큐에 담는다

3. 큐가 빌때까지 2를 계속 진행한다.

# [백준 - 골드 3] 사회망 서비스(SNS) (2533)

## ⏰  **time**

2시간(시간초과)

## :pushpin: **Algorithm**

DP

## ⏲️**Time Complexity**

$O(N)$

## :round_pushpin: **Logic**
1. 점화식

```
얼리어답터 최소 수[부모노드번호][얼리어탑터 맞음] = SUM(MIN(얼리어답터 최소 수[모든 자식노드번호][얼리어탑터 맞음],얼리어답터 최소 수[모든 자식노드번호][얼리어탑터 아님])) 
얼리어답터 최소 수[부모노드번호][얼리어탑터 아님] = SUM(얼리어답터 최소 수[모든 자식노드번호][얼리어탑터 맞음]) 
```

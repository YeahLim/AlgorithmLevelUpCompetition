# [백준 - 실버 4] 게임을 만든 동준이 (2847번)

## ⏰ **time**

30분

## :pushpin: **Algorithm**

그리디

## ⏲️**Time Complexity**

$O(n)$

## :round_pushpin: **Logic**

1. 끝에서 부터 배열을 돌며 만약 i-1이 i 이상이면 [i-1] - [i] + 1을 i-1에 빼주고, 카운트를 그만큼 증가시킨다.
   예를 들어 i가 5고 i-1이 7이면 4만 맞춰주면 되기때문에 이 식이 곧 위의 식과 같다.

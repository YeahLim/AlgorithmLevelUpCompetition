# [백준 - 실버 5] 그룹 단어 체커 (1316 번)

## ⏰  **time**

40분

## :pushpin: **Algorithm**

- 구현
- 문자열

## ⏲️**Time Complexity**

$O(N)$

## :round_pushpin: **Logic**

- 단어의 개수를 그룹 단어에 저장한 뒤 word 를 입력 받고 0번째 글자부터 for문을 돌렸다. 
- 연속된 글자가 같은 글자면 continue로 넘겼지만 다른 문자가 나온 이후 글자가 같은 글자가 나온다면 그것은 그룹 단어가 아니기 때문에 -1 해줬다.

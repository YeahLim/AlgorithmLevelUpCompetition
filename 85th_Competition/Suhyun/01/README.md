# [백준 - 골드 3] 나만 안되는 연애

## ⏰  **time**
15분

## :pushpin: **Algorithm**
MST(크루스칼 알고리즘)

## ⏲️**Time Complexity**
$O(N^2)$

## :round_pushpin: **Logic**
- 경로 중심으로 최단거리의 합을 구해야 하므로 크루스칼 알고리즘을 사용해야한다.
- 모두 연결되어 있는지 확인하기 위해 부모의 값을 모두 확인해서 모두 같은 값인지 확인한다.
- 남남, 여여로 연결되어 있는 경로는 미리 제거한다.

## :black_nib: **Review**
- 엣지 중심은 크루스칼 알고리즘

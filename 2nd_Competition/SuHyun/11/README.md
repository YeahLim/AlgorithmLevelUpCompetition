# [백준 - 골드5] 너의 평점은

## ⏰  **time**

40분

## :pushpin: **Algorithm**

DP

## ⏲️**Time Complexity**

$O(N)$

## :round_pushpin: **Logic**
1. 점화식
```java
   dp[현재 자리수(N)를 사용해서 나오는 경우의 수] 
        = dp[이전 자리수(N-1)와 현재 자리수(N)을 사용해서 나오는 경우의 수]
            + dp[이전 자리수(N-2)와 이전과 현재 자리수((N-1)*10 + N)을 사용해서 나오는 경우의 수]
        
     dp[N] = dp[N-1] + dp[N-2];
    이때 N이나 (N-1)*10 + N은 모두 1이상 26이하여야 한다.
```

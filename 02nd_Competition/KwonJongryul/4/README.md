# [백준] 보물 (1026번)

## ⏰ **time**

10분

## :pushpin: **Algorithm**

그리디

## ⏲️**Time Complexity**

O(n log n)

## :round_pushpin: **Logic**

문제에 'b는 정렬해선 안된다'에 낚일뻔했다.
어차피 A를 정렬후 B와 곱해서 더하는 이상 위의 전제는 의미가 없다.
A는 정배열, B는 역배열 후 곱해서 더해주면 된다.

## :black_nib: **Review**

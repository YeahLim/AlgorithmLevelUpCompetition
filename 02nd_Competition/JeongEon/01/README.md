# [백준 - 실버 3] 1448. 삼각형 만들기

## ⏰ **time**
10분

## :pushpin: **Algorithm**
그리디

## ⏲️ **Time Complexity**
$O(N \log N)$

## :round_pushpin: **Logic**
1. N개의 막대 길이를 배열에 저장한다.
2. 배열을 오름차순으로 정렬한다.
3. 가장 큰 막대 3개부터 차례대로 확인하며 삼각형 조건을 검사한다.
   * 삼각형 조건: 가장 긴 변 < 나머지 두 변의 합
4. 조건을 만족하는 **첫 번째 경우**가 곧 **둘레가 최대인 삼각형**이다.
5. 조건을 만족하는 조합이 하나도 없다면 `-1`을 출력한다.

```
if (arr[i+2] < arr[i+1] + arr[i]):
    최대 둘레 출력
else:
    다음 조합 확인
```

# [백준 - 골드 5] 2493. 탑

## ⏰  **time**

1시간

## :pushpin: **Algorithm**

스텍

## ⏲️**Time Complexity**

$O(NlogN)$

## :round_pushpin: **Logic**

1. 모든 탑은 데이터를 송신한다.
2. 하지만, 송신한 탑의 높이보다 크거나 같은 탑에서만 수신이 가능하다.
   - 스텍에 최대 높이와 0번 위치라고 넣는다.
   - 입력된 탑보다 크거나 같은 높이가 나올 때까지 스텍을 pop()한다.
   - 만약 입력된 탑보다 크거나 같은 높이가 나오면 그 위치를 출력하고, 입력된 탑의 높이가 위치를 stack에 push한다.

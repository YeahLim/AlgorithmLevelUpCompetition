# [백준 - 골드 2] 피자 판매 (2632)

## ⏰  **time**

20분

## :pushpin: **Algorithm**

누적합

## ⏲️**Time Complexity**

$O(N^2)$

## :round_pushpin: **Logic**
1. 문제풀이

각 조각의 연속된 조각의 합을 구하고(N^2) 목표 크기가 되는 짝이 있는지 확인(M) 하면 모든 경우의 수를 구할 수 있다.

이때 N의 최대 크기는 1,000이고, M의 최대 크기는 1,000,000 이기 때문에 목표 시간복잡도 안으로 계산이 가능하다

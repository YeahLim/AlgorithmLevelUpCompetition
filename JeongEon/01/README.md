# [백준 💛4️⃣] 1715. 카드 정렬하기
 
## ⏰  **time**

20분

## :pushpin: **Algorithm**

우선순위 큐

## ⏲️**Time Complexity**

$O(Nlog(N))$

## :round_pushpin: **Logic**

1. 우선순위 큐를 이용해 오름차순으로 계속 정렬한다.
2. 앞에서 두수를 더해 큐에 다시 push한다.
3. 더한 두수는 result에 더해준다.
4. 만약 큐의 크기가 1이하로 된다면 result값을 출력한다.

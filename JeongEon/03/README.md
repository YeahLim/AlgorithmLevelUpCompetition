# [백준 🩶4️⃣] 1302. 베스트셀러
 
## ⏰  **time**

12분

## :pushpin: **Algorithm**

정렬

## ⏲️**Time Complexity**

$O(Nlog(N))$

## :round_pushpin: **Logic**

1. 가장 많이 팔린 책의 제목을 출력한다.
2. 만약, 팔린 수가 같다면 제목의 오름차순으로 정렬 후 출력
3. sort compare함수를 따로 만들어, 팔린 수를 내림차순으로 정렬하고 같다면 제목의 오름차순으로 정렬하였다.

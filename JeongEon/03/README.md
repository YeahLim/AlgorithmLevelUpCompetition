$O(N^2)$

## :round_pushpin: **Logic**

과자는 반드시 2봉지를 골라야하므로 snack[[i] + snack[j] <= M 인 경우를 찾는다.
이때, 과자의 무게는 내림차순으로 정렬되어 있고, i = 0 (< N) j = N - 1(> i)

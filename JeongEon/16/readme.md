## :pushpin: **Algorithm**

## :round_pushpin: **Logic**

- 누적합 : 누적해서 더한 값으로 계산 (좌 + 위 - 좌상단 대각선 = 본인)
- 구간합 : K*K의 구간을 지정해서 누적합된 배열을 계산 (본인 = 본인 - 좌 - 위 + 좌상단 대각선)

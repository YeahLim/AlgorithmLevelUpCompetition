# [SWEA - D2] 12712. 파리퇴치3
 
## ⏰  **time**

50분

## :pushpin: **Algorithm**

구현

## ⏲️**Time Complexity**

$O(N^2)$

## :round_pushpin: **Logic**

1. 상하좌우 대각선 방향벡터를 이용해 sum에 더한다.
2. 꼭 범위 내여야지 sum에 더할 수 있다.
3. result에 sum을 담고, 내림차순으로 정렬한다.
4. result[0]을 출력한다.

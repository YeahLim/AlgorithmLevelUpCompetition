# [백준 - 골드 4] 그리드 네트워크

## ⏰  **time**
40분

## :pushpin: **Algorithm**
MST(크루스칼 알고리즘)

## ⏲️**Time Complexity**
$O(N^2)$

## :round_pushpin: **Logic**
- 경로 중심으로 최단거리의 합을 구해야 하므로 크루스칼 알고리즘을 사용해야한다.
- (x,y) 좌료를 평면화 하는 방법은 x * (열의 길이) + y

## :black_nib: **Review**
- 엣지 중심은 크루스칼 알고리즘
- 좌표를 평면화 하여 풀면 쉽다

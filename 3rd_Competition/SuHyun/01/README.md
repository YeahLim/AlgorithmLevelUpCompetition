# [백준 - 골드4] 수 묶기(1744)

## ⏰  **time**

10분

## :pushpin: **Algorithm**

그리디

## ⏲️**Time Complexity**

$O(N)$

## :round_pushpin: **Logic**
1. 문제 풀이

양수와 음수와 나누어서 생각한다.

- 짝을 이룰 수 있는 경우
  - 양수인 경우
      - 곱하는 것과 더하는 것 중 큰 것을 고른다. 
  - 음수 인경우
    - 곱하는 것이 양수가 되므로 무조건 더 크다.
- 짝을 이룰 수 없는 경우
  - 양수 인 경우
    - 더하면 무조건 수가 커지므로 더한다.
  - 음수인 경우
    - 0이 있는경우
      - 0이 있다면 곱하면 0이 되므로 음수보다 크다.
    - 0이 없는 경우
      - 0이 없다면 곱하는 것 보다 더한 것이 더 작다.

# [백준 - 골드 2] 스타트 택시 (19238)

## ⏰  **time**

2시간

## :pushpin: **Algorithm**

구현, BFS

## ⏲️**Time Complexity**

$O(N^2logN)$

## :round_pushpin: **Logic**
1. 구현 부분
   
```
   while(모든 승객을 다 목적지로 보낼 때 까지){
      모든 승객과 택시사이의 거리 계산
      
      택시와 승객거리 기준으로 정렬
      가장 거리가 짧은 승객 찾기
      
      택시를 승객의 위치로 이동
      만약 택시가 승객의 위치에 못가면{
         -1 출력
      }
      
      택시를 목적지에 최단거리로 이동
      
      만약 택시가 목적지에 못가면{
         -1 출력
      }
   }
```


## :black_nib: **Review**
- 미리 모든 노드간의 최단거리르 전부 구하는 것이 일일히 구하는 것 보다 빠를 때가 있다.

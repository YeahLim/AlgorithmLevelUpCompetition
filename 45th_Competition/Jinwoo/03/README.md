# [백준 - 실버 5] 숫자카드 (10815 번)

## ⏰  **time**

20 분

## :pushpin: **Algorithm**

- 자료 구조 
- 정렬 
- 이분 탐색 
- 해시를 사용한 집합과 맵

## ⏲️**Time Complexity**

$O(NlogN)$

## :round_pushpin: **Logic**

- query in cards를 통해 query가 cards에 포함되어 있는지 확인하는 방법으로 풀이를 진행하였다.

## :black_nib: **Review**

- 각 숫자에 대해 확인 결과를 리스트로 저장한 뒤, 공백으로 구분하여 한 번에 출력할 수 있었다.

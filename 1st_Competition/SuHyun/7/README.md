# [프로그래머스] 스타 수열

## ⏰  **time**

1시간(타임오버)

## :pushpin: **Algorithm**

그리디

## ⏲️**Time Complexity**

$O(N^2)$

## :round_pushpin: **Logic**
1. 문제 접근

집합 별 공통적인 것(교집합) = 0 ~ 숫자 최대값 내의 숫자가 각 집합에 모두 있는가

## :black_nib: **Review**
- 공통 된 것을 찾으려고 할 때 비교가 어렵다면 범위가 충분할 때 모든 범위의 수가 있는지 확인하는 방법도 생각해보자
- 시간 복잡도가 N^2이고 N이 500,000인데 제한시간을 안줘서 헷갈린 문제

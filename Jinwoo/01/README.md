# [백준 - 실버 5] CD (4518 번)

## ⏰  **time**

30분

## :pushpin: **Algorithm**

- 자료 구조 
- 이분 탐색 
- 집합과 맵 
- 해시를 사용한 집합과 맵 
- 두 포인터

## ⏲️**Time Complexity**

$O(N)$

## :round_pushpin: **Logic**

- 상근이와 선영이의 CD 번호는 이미 오름차순으로 정렬되어 있음 
- 두 개의 포인터 i, j를 각각 상근이 CD 리스트, 선영이 CD 리스트에 둔다 
- 두 포인터가 가리키는 값을 비교하면서 진행 
- 값이 같으면 → 공통 CD이므로 카운트 증가 후 두 포인터 모두 이동 
- 상근이 CD가 더 작으면 → 상근이 포인터 이동 
- 선영이 CD가 더 작으면 → 선영이 포인터 이동 
- 둘 중 하나라도 끝에 도달하면 종료 
- 각 테스트 케이스마다 결과 출력

## :black_nib: **Review**

- set으로 풀 수도 있지만, 최대 100만 개 입력이라 메모리 부담이 있었음

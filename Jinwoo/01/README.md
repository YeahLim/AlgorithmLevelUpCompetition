# [백준 - 실버 5] 크리 문자열 (11059 번)

## ⏰  **time**

30분

## :pushpin: **Algorithm**
 
- 브루트포스 알고리즘
- 누적 합

## ⏲️**Time Complexity**

$O(N)$

## :round_pushpin: **Logic**

- 문자열 S의 길이 n에서부터 2까지 내림차순으로 짝수 길이만 확
- 해당 길이의 모든 부분 문자열을 순회하며 절반씩 나눈 뒤, 각 부분의 자리 수 합을 비교 
- 조건을 만족하면 그 길이를 max_len에 저장하고 바로 종료 (가장 긴 것만 찾으므로)

# [백준 - 실버 4] 2+1 세일 (11508 번)

## ⏰  **time**

30분

## :pushpin: **Algorithm**

- 그리디 알고리즘
- 정렬

## ⏲️**Time Complexity**

$O(N log N)$

## :round_pushpin: **Logic**

- 유제품 가격을 내림차순 정렬한다.
- 앞에서부터 3개씩 묶으면서, 가장 비싼 2개만 합산, 나머지 하나는 무료 처리한다. 
- 남은 나머지 1~2개 제품은 정가로 더해준다.

## :black_nib: **Review**

- 최대한 비싼 것부터 먼저 묶어야 할인 혜택을 가장 크게 누릴 수 있었음

# [백준 - 실버 5] 단어 나누기 (1251 번)

## ⏰  **time**

30분

## :pushpin: **Algorithm**

- 구현 
- 문자열 
- 브루트포스 알고리즘 
- 정렬

## ⏲️**Time Complexity**

$O(K)$

## :round_pushpin: **Logic**

- 세 부분으로 나누기 
  - 인덱스를 2번 선택해서 단어를 3개 부분으로 나누되, 각 부분은 길이가 1 이상이어야 하므로 i는 1부터 len(s)-2, j는 i+1부터 len(s)-1까지 가능. 
- 각 부분 뒤집기 
  - 각 부분을 [::-1]로 뒤집고 합칩니다.
- 사전순으로 가장 앞서는 문자열을 선택 
  - 나올 수 있는 모든 경우를 리스트에 담아 min()을 취하면 됩니다.

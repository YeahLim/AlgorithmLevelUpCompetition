## :round_pushpin: **Logic**

- 각 반마다 학생 수와 점수 리스트를 입력받는다. 
- 점수 리스트를 내림차순 정렬한다. 
  - 최댓값 = 첫 번째 값 
  - 최솟값 = 마지막 값 
- 인접한 점수 차이 중 가장 큰 값을 구한다. 
- 문제에서 요구한 형식 그대로 출력한다.

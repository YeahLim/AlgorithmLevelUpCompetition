# [백준 - 실버 3] N번째 큰 수 (2075 번)

## ⏰  **time**

30분

## :pushpin: **Algorithm**

- 자료구조
- 정렬
- 우선순위 큐

## ⏲️**Time Complexity**

$O(N)$

## :round_pushpin: **Logic**

- 크기가 N인 최소 힙을 유지한다. 
- 힙에는 지금까지 본 수들 중 가장 큰 N개만 남긴다. 
- 힙의 크기가 N을 넘으면 가장 작은 값을 제거한다. 
- 그러면 마지막에 힙에는 전체 수 중 큰 수 N개만 남는다. 
- 이때 힙의 최솟값이 바로 N번째 큰 수이다.

# [백준 - 실버 5] 거스름돈 (14916 번)

## ⏰  **time**

30분

## :pushpin: **Algorithm**

- 수학
- 다이나믹 프로그래밍
- 그리디

## ⏲️**Time Complexity**

$O(N)$

## :round_pushpin: **Logic**

- 동전 개수를 최소로 만들기 위해 5원 동전을 최대한 사용한다.
- 남은 금액이 2로 나누어 떨어지면 2원 동전으로 채운다.
- 나누어 떨어지지 않으면 5원 대신 2원을 하나 사용하는 방식으로 반복한다.
- 끝까지 만들 수 없다면 -1 출력한다.

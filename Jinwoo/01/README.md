# [백준 - 실버 5] 암기왕 (1417 번)

## ⏰  **time**

20분

## :pushpin: **Algorithm**

- 구현
- 자료 구조
- 그리디 알고리즘
- 실물레이션
- 우선순위 큐

## ⏲️**Time Complexity**

$O(N)$

## :round_pushpin: **Logic**

- 다솜이를 제외한 다른 후보 중 가장 높은 득표자에게서 표를 하나씩 빼서 다솜이에게 넘긴다. 
- 이 작업을 반복하여 다솜이의 표가 가장 많아질 때까지 반복한다. 
- 이때 매수 횟수가 최솟값이 된다.

## :black_nib: **Review**

- 그리디 알고리즘을 활용하여
- 가장 많은 표를 가진 후보부터 표를 옮겨서 가장 빠르게 다솜이를 1등으로 만드는 것이 포인트

## ⏲️**Time Complexity**

$O(N)$

## :round_pushpin: **Logic**

- 첫번째 줄에 입력받을 명령어 갯수 K을 선언한다.
- for문을 사용하여 K만큼 입력을 받는다.
- 만일 입력이 0일 경우 pop(-1) 함수를 통해 마지막 값을 stack 리스트에서 빼준다.
- 0이 아닌 나머지 값들이 입력으로 들어올 경우 .append() 매서드를 이용하여 stack 리스트에 값을 추가한다.
- 반복문이 끝난 후 stack 리스트의 총합을 출력한다.

## :black_nib: **Review**

- 이번 문제는 정수가 0으로 입력되는 경우가 지울 수 있는 수가 있음을 보장한다는(stack에 값이 있다는걸 보장) 조건이 있어 쉽게 풀었다.
- pop() 매서드는 리스트의 마지막이나 처음값에 사용할 땐 시간복잡도가 O(1)이다.

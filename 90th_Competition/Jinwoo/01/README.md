# [백준 - 실버 2] 사사의 사차원 사탕 봉지 (27968 번)

## ⏰  **time**

30분

## :pushpin: **Algorithm**

- 브루트포스 알고리즘
- 백트래킹

## ⏲️**Time Complexity**

$O(N)$

## :round_pushpin: **Logic**

- 방문 여부 관리: visited 배열을 사용해 숫자 사용 여부를 기록한다. 
- 백트래킹:
  - 현재 경로의 길이가 N이 되면 출력
  - 그렇지 않으면 1부터 N까지 반복하며 아직 사용되지 않은 숫자를 선택하고 다음 단계로 재귀 호출 
- 백트래킹의 종료 조건은 현재 경로 길이가 N이 되었을 때이고, 이때 출력한다.

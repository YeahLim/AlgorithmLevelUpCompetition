# [프로그래머스 - level 2] 할인행사

## ⏰  **time**
20분

## :pushpin: **Algorithm**
슬라이드윈도우

## ⏲️**Time Complexity**
$O(N)$

## :round_pushpin: **Logic**
- 연속적으로 10일의 날짜에 있는 상품 중 내가 원하는 아이템이 모두 있는 경우를 구하면 됨
  - 이때 날짜의 범위가 이동할 땐 슬라이드 윈도우를 활용하여 첫날과 마지막날 값만 바꾸면 N의 시간복잡도로 해결이 가능함

## :black_nib: **Review**
- get을 이용하면 딕셔너리에 키가 없을때의 초기값을 리턴할 수 있음

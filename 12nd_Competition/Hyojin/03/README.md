## 📍 Logic
arr[4] = arr[3] + arr[2] 

열의 길이가 4가 되는 경우는 (열의 길이가 총 3이 되는 경우에 1인 도형을 넣는 경우) + (열의 길이가 총 2인 경우에 2인 도형을 넣는 경우)라서

arr[n] = arr[n-1] + arr[n-2]

배열 크기 최대로 설정해주기 주의

# [SWEA D4️⃣] 1218. [S/W 문제해결 기본] 4일차 - 괄호 짝짓기
 
## ⏰  **time**

17분

## :pushpin: **Algorithm**

스텍

## ⏲️**Time Complexity**

$O(N)$

## :round_pushpin: **Logic**

각 괄호 별로 스텍을 만들어 넣는다.
각 여는 괄호는 각 스텍에 넣고, 닫는 괄호는 empty가 아니라면 pop을 하고 아니면 result = 0을 출력한다.
모든 괄호를 탐색 후 모든 스텍이 empty가 아니라면 result = 0을 출력한다.

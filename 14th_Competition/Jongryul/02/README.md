# [백준 - 실버 3 ] 합 구하기 (11441번)

https://www.acmicpc.net/problem/11441

## ⏰ **time**

20분

## :pushpin: **Algorithm**

구간합

## ⏲️**Time Complexity**

$O(N)$

## :round_pushpin: **Logic**

1. 구간합 기본문제이다. 누적합을 구한 후, 시작이 첫번째 인덱스가 아니라면 끝 값에 시작 전 인덱스의 값을 빼준다.

## :black_nib: **Review**

- 구간합도 까먹을까봐 오랜만에 풀었다.
